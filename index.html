<h1>

Tags h1-h6 are used to define headings.

* Tip; Only use one h1 per page.
<p>

Used to define groups of text.

* Tip; "p" stands for paragraph.
<br>

The break tag breaks up elements.

* Tip; br is a self-closing tag. You only need one!
<hr>

The horizontal rule tag adds a break and line.

* Tip; hr is a self-closing tag. You only need one!
