color

Defines color of the foreground element.

* Tip; There are 3 methods to control color.

margin

Defines the space around elements.

* Tip; You can set margins for all 4 sides.
padding

Defines space inside content borders.

* Tip; Padding keeps elements safe in their box.

float

Controls where the elements sits in the flow.

* Tip; You can float; left, right, none or inherit.
